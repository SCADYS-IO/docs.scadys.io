# Practical Compliance Test Plan Template

This template provides a structured approach for preparing and executing compliance testing for SCADYS.io NMEA 2000 products. It should be adapted for each specific product.

---

## 1. Purpose

Describe the aim of this plan — to outline test setup, sequence, and methods that will verify compliance with applicable standards.

## 2. Applicable Standards

List the compliance standards and directives relevant to the product (e.g., EN 60945, FCC Part 15B, AS/NZS CISPR 32, IEC 60529).

## 3. Test Categories

* **EMC – Emissions**: Conducted and radiated emissions.
* **EMC – Immunity**: Radiated immunity, conducted immunity, ESD, EFT, surge.
* **Environmental**: Temperature, humidity, vibration, salt mist.
* **Ingress Protection**: Water and dust ingress.
* **Safety**: Applicable electrical safety checks.
* **Chemical**: RoHS and REACH conformity.

## 4. Test Setup

Describe:

* Test locations (e.g., in-house shed, external pre-compliance lab, accredited compliance lab).
* Equipment and accessories used.
* Power source configuration (e.g., 12 V battery vs bench PSU).
* DUT operating mode for each test.

## 5. Test Sequence

Outline the order of tests. Example:

1. Visual inspection and documentation.
2. Functional baseline tests.
3. Conducted emissions.
4. Radiated emissions.
5. Immunity tests.
6. Environmental and ingress tests.
7. Final functional verification.

## 6. Acceptance Criteria

State the limits and conditions for passing each test type. Reference the relevant clauses of the applicable standards.

## 7. Data Recording

* Measurement methods and formats (e.g., tabulated results, spectrum plots, annotated images).
* File storage and naming conventions.
* Calibration requirements for test equipment.

## 8. Contingency Actions

Describe steps to follow if a test fails, including troubleshooting, mitigation, and retest procedure.

## 9. Deliverables

List the outputs from executing this plan:

* Test reports.
* Updated technical documentation.
* Compliance evidence for certification.

## 10. References

List linked documents, such as the relevant **Compliance Test Strategy Template** and per-product specifications.
