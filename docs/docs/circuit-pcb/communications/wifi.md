# Wifi

Wi-Fi is reserved for future expansion to support higher-level marine data protocols such as:

* **SignalK**: Open-source data format and APIs for marine applications
* **OneNet**: NMEA standard for transmitting NMEA 2000 messages over Ethernet/IP

While Wi-Fi is not currently active in production firmware, hardware support is present and reserved for future features that require high-throughput or IP-based integration.
