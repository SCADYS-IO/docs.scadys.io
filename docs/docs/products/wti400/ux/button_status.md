# Button and Status LED

WTI400 provides a local user button and multi-color status LED for commissioning feedback and runtime diagnostics.

## User Button

The local button is used for mode transitions such as setup entry and maintenance actions.

!!! note
	Electrical behavior and GPIO mapping for the button are defined in the [WTI400 quick reference](../quick_reference.md).

## Status LED

The RGB indicator communicates device state and fault conditions during startup, commissioning, and normal operation.

!!! note
	LED electrical mappings are documented in the [WTI400 quick reference](../quick_reference.md).

Detailed blink/state definitions will be expanded as firmware behavior is finalized.
