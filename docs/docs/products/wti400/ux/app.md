# Bluetooth and App Workflow

WTI400 supports app-assisted setup for commissioning and maintenance tasks.

## Typical Workflow

1. Power the device and enter commissioning mode.
2. Connect from a supported mobile app over Bluetooth.
3. Select transducer profile and apply calibration values.
4. Verify live signal behavior and network output.
5. Save configuration and return to normal operating mode.

## Intended Scope

App interaction is intended for setup, diagnostics, and controlled parameter updates. Primary operation remains autonomous after commissioning.
