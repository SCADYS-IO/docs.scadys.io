# Compliance Overview

This section summarizes regulatory and standards targets for the WTI400 analog wind transducer interface.

WTI400 is intended for marine use and designed for interoperability with both modern and legacy marine data networks. Compliance work is tracked against product category, electrical interface, and deployment environment requirements.

Covered areas include:

* CE/UKCA and FCC EMC pathways for electronic equipment;
* protocol-aligned operation on NMEA 2000 and compatibility with legacy NMEA 0183 / SeaTalk interfaces;
* environmental and material directives (for example RoHS, REACH, and WEEE where applicable);
* enclosure and ingress-protection expectations for marine installation contexts.

## Status

WTI400 remains in the pre-production documentation phase. Final third-party certification and market-specific declarations are completed as part of release readiness and manufacturing handover.
