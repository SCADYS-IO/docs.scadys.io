# Temperature Sensor

Temperature monitoring is used to provide insights into internal thermal conditions that may affect reliability, performance, and user experience. The WTI400 uses the internal ESP32-S3 junction sensorto provide visibility into the thermal state of the device, improving resilience and field reliability.

!!! note
    Refer to the [WTI400 quick reference](../../quick_reference.md) for GPIO and I2C bus address assignments.
