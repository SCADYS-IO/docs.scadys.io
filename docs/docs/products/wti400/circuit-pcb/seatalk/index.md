# _LEGACY IO_ Domain

On selected models, the WTI400 includes a plug-and-play serial interface that supports legacy marine protocols such as NMEA 0183 and SeaTalk<sup>®</sup> I.  

!!! note
    The `ST_RX`, `ST_TX` and `ST_EN` GPIO pin allocations for the interface are provided in the [WTI400 quick reference](../../quick_reference.md).
