# SCADYS.IO Technical Documentation

This site provides a detailed engineering reference for the SCADYS.IO product family, including the MDD400 Marine Data Display and WTI400 Analog Wind Transducer Interface.

It is intended for use by engineers, developers, and technical reviewers involved in the design, testing, integration, and certification of SCADYS.IO products. The documentation includes full schematics, PCB design descriptions, firmware architecture, and compliance information.
