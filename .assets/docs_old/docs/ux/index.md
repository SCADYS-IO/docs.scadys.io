# UX

This section outlines the user experience and interaction design of the MDD400. It focuses on the presentation layer, touch interactions, and mobile app integration used for configuration, monitoring, and diagnostics.

* *[Display & Touch](display.md)*: Describes the visual layout, touchscreen behavior, and interaction flow on the LCD. Covers menu navigation, gesture input, and responsiveness under varied lighting and environmental conditions.

* *[Bluetooth & App](app.md)*: Documents the Bluetooth-based configuration workflow using a mobile companion app. Includes commissioning process, parameter editing, and firmware update via BLE.

* *[Appendices](appendices.md)* and *[Change Log](changelog.md)*: Supporting material and revision history related to user interface features.

This section provides context for designing, evaluating, and extending the MDD400’s user interaction model.
